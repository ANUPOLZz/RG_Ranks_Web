INSTALLATION GUIDE

1

Download the Essentials

For Java, first download the Java resources pack. For Bedrock, download the Geyser and Floodgate plugin in your server.

2

File Configuration

For Java, host your java resources pack through https://mc-packs.net/ , For Bedrock, Restart/Start Your Server So New files will be generated.

3

Pack Configuration

For Java, copy the pack Sha1 and resources pack link that you got from hosting resources pack in mcpacks, For Bedrock, after the server restart/start is done, go to your server files, then plugins, then find geyser folder, open it and go to packs folder and if there no such folder, then create one with the name 'packs'.

4

Pack Setup

For Java, open your server panel, then go to files and find 'server.properties' and inside it, find the 'resource-pack= ' and paste the link there, then again, find the 'resource-pack-sha1= ' and paste the sha1 you got there. Then restart/start your server. For Bedrock, upload the Bedrock/MCPE resources pack there and restart/start your server.
