INSTALLATION GUIDE

1

Download Files

For Java, download the resource pack and keep it ready locally; for Bedrock, download Geyser and Floodgate and place them in your server’s plugins folder (or upload them via your server panel).

2

Prepare Files

For Java, upload the resource pack to MC-Packs to host it and note the pack URL and SHA-1; for Bedrock, start the server once so Geyser generates its config files in plugins/Geyser.

3

Configure Pack

For Java, copy the hosted pack URL and SHA-1 hash from MC-Packs; for Bedrock, open your server files, go to plugins/Geyser and create a packs folder at /plugins/Geyser/packs if it doesn’t exist.

4

Apply Pack

For Java, edit server.properties to set resource-pack= (paste the URL) and resource-pack-sha1= (paste the SHA-1) then save and restart your server via the panel; for Bedrock, upload the pack to plugins/Geyser/packs (or your server’s packs folder) and restart.
